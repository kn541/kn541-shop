## 6. 발생 이슈 및 해결

| No | 이슈 | 원인 | 해결 |
|----|------|------|------|
| 1 | next-intl 설치 실패 | next-intl@3.x가 Next.js 16 미지원 | next-intl@^4.4.0으로 업그레이드 |
| 2 | TypeScript 빌드 오류 | Header 컴포넌트에 없는 hasBorderBottom prop | layout.tsx 2개에서 prop 제거 |
| 3 | middleware 경고 | Next.js 16에서 middleware.ts 폐기됨 | proxy.ts 생성, middleware.ts 비움 |
| 4 | /categories API 401 | 공개 조회 API에 인증 의존성 존재 | get_current_user 제거 |
| 5 | 입력박스 스타일 일부 미반영 | search/page.tsx 하드코딩 rounded-full | 직접 rounded-xl 수정 |
