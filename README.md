# KN541 쇼핑몰 프론트

Ciseco (Next.js 15 + TypeScript + Tailwind CSS 4) 기반 KN541 쇼핑몰

## 기술스택

- **프레임워크**: Next.js 15 (App Router)
- **언어**: TypeScript
- **스타일**: Tailwind CSS 4
- **배포**: Vercel
- **백엔드**: https://kn541-production.up.railway.app

## 시작하기

```bash
npm install
npm run dev
```

## 환경변수 설정

`.env.local` 파일 생성:

```
NEXT_PUBLIC_API_URL=https://kn541-production.up.railway.app
NEXT_PUBLIC_SHOP_DOMAIN=kn541shop.com
```
