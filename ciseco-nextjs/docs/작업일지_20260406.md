## 2. 상세 내용

### 장바구니 슬라이드 팝업 다국어 적용

**문제:** 탑 쇼핑카트 아이콘 클릭 시 우측에 열리는 슬라이드 팝업이 언어 선택(한국어/영어/중국어)에 관계없이 항상 영문으로 표시됨

**원인:** `aside-sidebar-cart.tsx` (Next.js Server Component)에 모든 UI 텍스트가 영문 하드코딩

```tsx
// 수정 전 — 영문 하드코딩
<Aside heading="Shopping Cart">
  <p>Subtotal</p>
  <ButtonSecondary>View cart</ButtonSecondary>
  <ButtonPrimary>Check out</ButtonPrimary>
  <p>or Continue Shopping</p>
  <button>Remove</button>
```

```tsx
// 수정 후 — next-intl getTranslations 적용
const t = await getTranslations('Cart')
<Aside heading={t('title')}>
  <p>{t('subtotal')}</p>
  <ButtonSecondary>{t('viewCart')}</ButtonSecondary>
  <ButtonPrimary>{t('checkout')}</ButtonPrimary>
  <p>{t('or')} {t('continueShoppingLink')}</p>
  <button>{t('remove')}</button>
```

**수정 파일:**
- `ciseco-nextjs/src/components/aside-sidebar-cart.tsx` — `getTranslations('Cart')` 적용
- `ciseco-nextjs/messages/ko.json` — Cart 섹션 번역키 추가
- `ciseco-nextjs/messages/en.json` — Cart 섹션 번역키 추가
- `ciseco-nextjs/messages/zh.json` — Cart 섹션 번역키 추가

**추가된 번역 키:**

| 키 | 한국어 | 영어 | 중국어 |
|----|--------|------|--------|
| `title` | 장바구니 | Shopping Cart | 购物车 |
| `subtotal` | 소계 | Subtotal | 小计 |
| `viewCart` | 장바구니 보기 | View cart | 查看购物车 |
| `checkout` | 결제하기 | Check out | 去结算 |
| `or` | 또는 | or | 或 |
| `continueShoppingLink` | 쇼핑 계속하기 | Continue Shopping | 继续购物 |
| `orderSummary` | 주문 요약 | Order summary | 订单摘要 |
| `shippingTaxNote` | 배송비 및 세금은 결제 시 계산됩니다. | Shipping and taxes calculated at checkout. | 运费和税费将在结算时计算。 |
| `remove` | 삭제 | Remove | 删除 |
